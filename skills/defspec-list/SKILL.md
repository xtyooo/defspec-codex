# DefSpec List

Summarize current project DefSpec requirements.

## Steps

1. Read `docs/defspec/requirements/index.md`.
2. Summarize requirements by status, newest first.
3. Mention active `draft`, `confirmed`, `in_progress`, and `dev_completed` items first.
4. If the user asks for details, read the relevant `REQ-xxx` files.
5. Do not modify files.
